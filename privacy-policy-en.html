1. Introduction

"Google Meet Recording Reminder" (the "Extension") is a Chrome extension designed to prevent users from forgetting to start recording during Google Meet sessions. This Privacy Policy explains how the Extension handles user information.

2. Information Collection

This Extension does not collect any personal information from users.

  • No personal identifiable information such as names, email addresses, or phone numbers is collected
  • Meeting content or participant information is not collected
  • Recording data itself is not collected
  • No data is transmitted to external servers

3. Permission Usage

The Extension uses the following permissions solely for functionality purposes and not for information collection:

3.1 host_permissions (meet.google.com/*)

  • Purpose: Execute functionality on Google Meet pages
  • Function: Recording button detection and warning display only
  • Data Storage: None

4. Data Storage and Processing

The Extension processes data as follows:

  • All recording status monitoring is performed locally within the browser
  • Meeting start times are temporarily stored in memory only
  • All temporary data is deleted when leaving the page
  • No persistent data storage is performed

5. Third-Party Information Sharing

The Extension does not share any information with third parties. All processing is completed within the user's browser.

6. Security

The Extension implements the following security measures:

  • Requests only minimum necessary permissions
  • Operates only on Google Meet pages
  • No external communication whatsoever
  • Safe design compliant with Manifest V3

7. User Rights

Since the Extension does not collect data, there is no need for data deletion or modification, but users have the following rights:

  • Extension can be disabled or removed at any time
  • Extension permissions can be individually controlled in browser settings
  • Questions about this policy can be sent at any time

8. Policy Changes

This Privacy Policy may be updated as necessary. Important changes will be announced in the Chrome Web Store extension description.

9. Governing Law

This Privacy Policy is governed by Japanese law, and Japanese courts have exclusive jurisdiction.
